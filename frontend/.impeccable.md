# SignalCraft Biz Design Context

## Brand Identity
- **Users**: Korean food/frozen facility managers, mobile-first field use
- **Brand feel**: Trust, professionalism, approachability (Toss-style)
- **Aesthetic**: Premium Korean fintech (Toss/KakaoBank) — clean white + blue neutrals
- **Anti-patterns**: Generic SaaS dashboards, glassmorphism abuse, dark-mode AI aesthetic

## Design Tokens
- **Primary**: oklch(0.62 0.2 255) (Signal Blue)
- **Typefaces**: Plus Jakarta Sans (headings) + Pretendard (body)
- **Radius scale**: 12px (sm), 16px (md), 24px (lg)
- **Spacing**: 4px base scale
- **Motion**: Purposeful entry transitions only, no infinite animations
